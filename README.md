# 🚀 Future of Transportation using Augmented Reality

An Android AR app that overlays futuristic vehicle animations 
onto real-world marker images in real time.

## 📱 Vehicles Featured
- 🚗 AI Electric Car
- ⛵ AI Solar Boat  
- 🚄 Bullet Train
- 🌀 Hyperloop System
- ✈️ Solar Aircraft
- 🚀 Space Travel System

## 🛠️ Built With
- Unity 2022 LTS
- AR Foundation + Google ARCore
- Blender (original 3D animations)
- C#

## ▶️ How to Run
1. Clone the repo in Unity 2022 LTS
2. Add MP4 videos to Assets/AssetsVideos/
3. Build to Android device
